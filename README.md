# TrackIt

flutter app

To run this app, you need to have Flutter installed on your computer.
If you haven't installed Flutter, follow the official installation guide for your platform: [Flutter Installation Guide](https://docs.flutter.dev/install).

## Installation

#Clone the repository to your local machine:

`git clone https://github.com/ujwalnk/Ritual.git`


#Navigate to the project directory:

`cd Ritual`


#Install the project dependencies:

`flutter pub get`


#Run the application

`flutter run`
